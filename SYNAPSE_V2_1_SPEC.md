# SYNAPSE_V2_1_SPEC.md

Implemented in this archive: **Patch 2.1.0 — Affective Memory Layer**.

The approved v2.1 specification defines four subpatches. This release deliberately implements only 2.1.0 to keep replay, memory and VM changes isolated.

Included: affective_tag, affective_decay, affective_filter, affective_sort, affective_routing, dual decay metadata, affective memory metrics, and regression tests.

Deferred to later subpatches: CVM checkpoint/resume (2.1.1), reactive thresholds (2.1.2), living habits and energy pool (2.1.3).
