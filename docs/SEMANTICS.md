# Operational Semantics Tables
Строгие runtime-контракты для когнитивных и аффективных примитивов.

| Примитив | Runtime Effect | Durable Events | Replay Behavior | Forbidden Contexts | Energy/Gas Cost |
|----------|----------------|----------------|-----------------|--------------------|-----------------|
| `habit "X" from pattern` | Регистрирует body в HabitRegistry, metadata в palace.procedural. O(1) подписка на события. | `habit_registered`, `habit_candidate_suggested`, `habit_activated`, `habit_suppressed`, `habit_fatigued`, `habit_recovered`, `habit_execution_failed` | В REPLAY оценка условий и исполнение body пропускаются. Состояние восстанавливается из событий. | Рекурсивный вызов той же привычки, глубина вложенности > 3, выполнение в REST mode при недостатке энергии. | `energy_cost` × multiplier если FATIGUED. Списывается только при успехе. Refund при ошибке. |
| `affective threshold "X"` | Регистрирует PAD-условие, sliding window, cooldown. При срабатывании выполняет action с подавлением observers. | `affective_threshold_triggered`, `affective_threshold_cleared`, `affective_threshold_action_failed` | В REPLAY условие не перевычисляется. Runtime читает `triggered` из history и восстанавливает состояние. | Внешние side-effects в action: `send`, `migrate`, `memory.write/forget`, `declare intent`, прямой `imprint`. | 0 как внутренняя реакция. Action может триггерить gas/energy если вызывает другие примитивы, ограничено purity. |
| `fracture self into` | Создаёт изолированные sub-agents с shadow soulprint, замораживает base mailbox, запускает consensus/integrate. | `identity_fractured`, `subagent_terminated`, `identity_integrated`, `fracture_panic` | Ephemeral compaction: если в history есть пара `fractured` → `integrated`, sub-agent logic пропускается. Применяется итог. | Вложенность > 2, `integrate` внутри sub-agent, вызов `affective resonance` внутри sub-agent. | Высокий: LLM calls, изоляция окружения. Учитывается CVM gas если скомпилировано. |
| `dream { ... } integrate { ... }` | Запускает изолированный sandbox с inference. Мутации запрещены. При успехе применяет integrate транзакционно через `StateOverlay`. | `dream_completed` (с `dream_key`, `result_hash`, `nested_event_policy`), `dream_isolation_violation`, `integrate_committed` (с `pre_state_hash`, `post_state_hash`, `write_set`, `write_set_hash`), `integrate_aborted` (с `abort_reason`, `pre_state_hash`, `overlay_summary`) | Alpha3g I1–I6 (Path A + replay-applier): dream body исполняется для синхронизации курсора, `dream_completed` потребляется с верификацией `dream_key`/`result_hash`. Integrate body в REPLAY **не исполняется** — `replay_integrate_i4_event` потребляет `integrate_committed`, верифицирует `pre_state_hash`/`write_set_hash`/`post_state_hash`, применяет записанный write-set. In-run idempotency guard. | `send`, `migrate`, `memory.write/forget/clear`, `evolve`, `fracture`, `affective resonance`, LLM calls (явно запрещены в integrate). | Высокий: LLM sampling/temperature. Gas-metered в CVM. |
| `affective resonance with @user` | Атомарно вычисляет и применяет PAD deltas: mirror/regulate/dampen. Обновляет AffectiveState одной транзакцией. | `affective_resonance_applied` с флагом `atomic: true` и списком deltas. | Вычисления пропускаются. Дельты применяются напрямую из исторического события. Итоговый PAD идентичен LIVE. | Внутри `dream`, внутри `fracture` sub-agent. В `integrate` разрешён как read-only/cache-safe. | Низкий: PAD math. Триггерит переоценку thresholds. |
